Comandos Git & Github

01

Incializar Proyecto

mkdir proyecto-git
cd proyecto-git
git init
02

Primer commit local

git add.
git commit -m "proyecto base"
03

Crear repo en github (web)

Crear un repositorio en Github

04

Vincular con la nube (github)

git remote add origin URL_DEL_REPO_REMOTO
05

Subir al repositorio remoto con push

git push -u origin master
06

Actualización (Nuevos cambios)

git add .
git commit -m "nuevo cambio"
git push origin master
